## 4. 常驻服务

| 服务 | 作用 | 状态 |
| --- | --- | --- |
| `agent-pilot-api.service` | 运行 Agent API | active |
| `agent-pilot-lark-events.service` | 运行飞书长连接事件桥接 | active |
| `nginx.service` | 提供 HTTPS、静态页面和反向代理 | active |

常用命令：

```bash
systemctl status agent-pilot-api.service
systemctl status agent-pilot-lark-events.service
systemctl status nginx.service
journalctl -u agent-pilot-api.service -n 100 --no-pager
journalctl -u agent-pilot-lark-events.service -n 100 --no-pager
```

重启：

```bash
systemctl restart agent-pilot-api.service agent-pilot-lark-events.service nginx.service
```

## 5. HTTPS 与域名策略

当前没有购买正式域名，采用：

```text
agent-pilot.47-236-122-49.sslip.io
```

该域名解析规则由 `sslip.io` 提供：

```text
agent-pilot.47-236-122-49.sslip.io -> 47.236.122.49
```

证书已通过 Let's Encrypt 签发，有效期到 `2026-07-30`，Certbot 已安装自动续期任务。

如果后续购买正式域名，只需要：

1. 在域名服务商添加 `A` 记录指向 `47.236.122.49`。
2. 修改 Nginx `server_name`。
3. 重新执行 Certbot 签发正式域名证书。
4. 把飞书后台 URL 从 `sslip.io` 替换成正式域名。

## 10. 安全提醒

当前为了快速比赛演示，使用 root 完成初始化部署。后续建议：

1. 立即修改 root 密码。
2. 创建独立部署用户。
3. 添加 SSH key 登录。
4. 禁用 root 密码登录。
5. 定期备份 `/opt/agent-pilot/.env`、`/opt/agent-pilot/.data` 和 `/root/.lark-cli/config.json`。
